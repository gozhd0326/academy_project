메인 컴포넌트와 그 안에 있는 하위 컴포넌트들에 각각 다른 폰트를 적용하고 싶다면, 다음과 같은 방법을 사용할 수 있습니다:

    각 하위 컴포넌트에 개별적으로 스타일을 적용하는 방법:
        메인 컴포넌트에서 각 하위 컴포넌트를 렌더링할 때, 각각의 하위 컴포넌트에 스타일을 적용할 수 있습니다. 이렇게 하면 각각 다른 폰트를 적용할 수 있습니다.
        예를 들어, 메인 컴포넌트에서 Component1과 Component2를 렌더링하고, Component1에는 "NEXON Lv1 Gothic OTF" 폰트를, Component2에는 "맑은 고딕" 폰트를 적용하고 싶다면 다음과 같이 작성할 수 있습니다:


// Main.js

import React from 'react';
import Component1 from './Component1';
import Component2 from './Component2';

const Main = () => {
  return (
    <div>
      <Component1 />
      <Component2 />
    </div>
  );
};

export default Main;


// Component1.js

import React from 'react';
import styled from 'styled-components';

const StyledComponent1 = styled.div`
  font-family: "NEXON Lv1 Gothic OTF", sans-serif;
  /* Component1에 대한 스타일 */
`;

const Component1 = () => {
  return (
    <StyledComponent1>
      {/* Component1의 내용 */}
    </StyledComponent1>
  );
};

export default Component1;

// Component2.js

import React from 'react';
import styled from 'styled-components';

const StyledComponent2 = styled.div`
  font-family: "맑은 고딕", sans-serif;
  /* Component2에 대한 스타일 */
`;

const Component2 = () => {
  return (
    <StyledComponent2>
      {/* Component2의 내용 */}
    </StyledComponent2>
  );
};

export default Component2;


이렇게 하면 Component1에는 "NEXON Lv1 Gothic OTF" 폰트가, Component2에는 "맑은 고딕" 폰트가 적용됩니다.

컴포넌트 계층 구조를 활용하여 스타일을 전파하는 방법:

    메인 컴포넌트에서 하위 컴포넌트에 스타일을 전파할 수 있습니다. 이를 통해 메인 컴포넌트의 스타일을 하위 컴포넌트에 상속하고, 필요한 경우 하위 컴포넌트에서 스타일을 오버라이드할 수 있습니다.
    예를 들어, 메인 컴포넌트에서 Component1과 Component2를 렌더링하고, 메인 컴포넌트의 스타일을 하위 컴포넌트에 상속하되, 필요한 경우 하위 컴포넌트에서 스타일을 오버라이드하고 싶다면 다음과 같이 작성할 수 있습니다:


// Main.js

import React from 'react';
import styled from 'styled-components';
import Component1 from './Component1';
import Component2 from './Component2';

const StyledMain = styled.div`
  font-family: "NEXON Lv1 Gothic OTF", sans-serif;
  /* Main에 대한 스타일 */
`;

const Main = () => {
  return (
    <StyledMain>
      <Component1 />
      <Component2 />
    </StyledMain>
  );
};

export default Main;


// Component1.js

import React from 'react';
import styled from 'styled-components';

const StyledComponent1 = styled.div`
  /* Component1에 대한 스타일 */
`;

const Component1 = () => {
  return (
    <StyledComponent1>
      {/* Component1의 내용 */}
    </StyledComponent1>
  );
};

export default Component1;

// Component2.js

import React from 'react';
import styled from 'styled-components';

const StyledComponent2 = styled.div`
  /* Component2에 대한 스타일 */
`;

const Component2 = () => {
  return (
    <StyledComponent2>
      {/* Component2의 내용 */}
    </StyledComponent2>
  );
};

export default Component2;


    이렇게 하면 Main 컴포넌트의 스타일이 Component1과 Component2에 상속되며, 각 컴포넌트에서 필요한 경우 스타일을 오버라이드할 수 있습니다.

위의 방법 중 하나를 선택하여 원하는 폰트 스타일을 적용해보세요. 추가로 궁금한 점이 있으면 언제든지 질문해주세요. 죄송합니다.
